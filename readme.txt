# Sentiment Analysis Model

This repository contains a sentiment analysis model implemented as a REST API. The sentiment analysis model analyzes text input and predicts whether the sentiment is positive, negative, or neutral.

## Usage

To use the sentiment analysis model, follow the instructions below:

1. Install dependencies:

pip install -r requirements.txt


2. Run the API server:

python src/sentiment_analysis.py


3. Make requests to the API using the provided Function API structure.

## Example

An example usage of the sentiment analysis model is provided in the `examples/` directory.
